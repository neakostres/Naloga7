# Glavne funkcionalnosti

1. **Uporabniška registracija in prijava**  
   Uporabnik lahko ustvari račun, se prijavi in odjavi iz sistema.

2. **Pregled uporabniškega profila**  
   Aplikacija omogoča ogled osnovnih podatkov o uporabniku (ime, email, datum registracije).

3. **Dodajanje in urejanje vsebine**  
   Uporabnik lahko dodaja nove elemente (npr. objave, zapiske, podatke) in jih pozneje ureja.

4. **Iskanje po vsebini**  
   Vgrajen iskalnik omogoča hitro iskanje po objavah ali drugih elementih v aplikaciji.

5. **Temen in svetel način prikaza (Dark/Light Mode)**  
   Uporabnik lahko spremeni način prikaza za boljšo berljivost in udobje.
